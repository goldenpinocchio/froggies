Froggies

Froggies is a Bitcoin-native pond collection designed for nat.fun, with a fixed supply of 4,032 unique Froggies. Supply, generations, and traits are derived entirely from Bitcoin blockchain data.

2qualifying fc difficulty periods

Supply generation

Bitcoin adjusts mining difficulty every 2,016 blocks. Froggies uses difficulty periods whose block header bits value contains the fc signal.

Trait generation

Every Froggie is linked to one Bitcoin block. Its block hash is split into deterministic rolls, then mapped into rarity ranges for each visible trait category.

01

Assign block

Each Froggie receives one block from a qualifying fc difficulty period.

02

Split hash

The 64-character Bitcoin block hash is divided into deterministic hexadecimal chunks.

03

Convert rolls

Each chunk becomes a number used as a repeatable trait roll.

04

Map traits

Rolls are mapped to rarity ranges to generate the final Froggie attributes.

Hash logic

The same block hash always produces the same Froggie. Anyone can reproduce the result from on-chain data alone.
