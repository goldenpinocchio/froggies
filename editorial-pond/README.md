# Variant: Editorial Pond

## Design stance
Flat vector Frogies inspired by Scandinavian minimal illustration, modern children’s books, and calm editorial art.

## Key choices
- Layout: large nature scene on the left, trait controls on the right
- Typography: soft, readable system type with a calm editorial hierarchy
- Color: deep teal base with muted sage, pink, and dusty blue accents
- Interaction: trait buttons update the frog’s body, eyes, prop, and background

## Trade-offs
- Strong at: capturing the reference image’s flat, calm, geometric feel
- Weak at: less glossy and less “toy collectible” than the earlier POCs

## Best for
- The main Frogies visual direction if you want the collection to feel quieter, more art-directed, and more premium
